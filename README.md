
# Classroom

Herramienta Full Stack construida con el stack MERN.


## Tech Stack

**Client:** React, Redux, Styled Components 

  
## Ejecucion local

Clonar el proyecto.

```bash
  git clone https://github.com/julandrod/Classroom-Client.git
```

Ir al directorio client

```bash
  cd my-project
```

Instalar dependencias

```bash
  npm install
```

Iniciar el cliente

```bash
  npm start
```
